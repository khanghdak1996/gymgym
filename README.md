# GymPair 🏋️

Ứng dụng cá nhân theo dõi quá trình tập luyện Gym với giao diện Dark Mode cao cấp, tích hợp bộ đếm giờ nghỉ thông minh (có âm thanh và rung báo hiệu) và theo dõi thông số tập luyện (Weight, Reps, RPE).

## 🚀 Các tính năng chính
- **Hồ sơ cá nhân:** Hỗ trợ chuyển đổi hồ sơ giữa "Anh" (💪) và "Em" (🌸) với màu sắc chủ đề tương ứng.
- **Thống kê Dashboard:** Xem tổng số buổi tập, tổng volume tạ nâng và số buổi tập trong tuần.
- **Bộ đếm giờ nghỉ thông minh (Radial Rest Timer):** Vòng tròn đếm ngược đè nổi màn hình, tự động phát âm thanh cảnh báo và rung khi hết giờ.
- **Lưu lịch sử tập luyện:** Thống kê lịch sử các buổi tập, xem lại chi tiết từng bài, số set, khối lượng tạ, chỉ số RPE và ghi chú.

## 📦 Cách cài đặt & Sử dụng
Bạn chỉ cần mở trực tiếp file `index.html` trên trình duyệt máy tính hoặc điện thoại di động để sử dụng (app chạy hoàn toàn offline thông qua LocalStorage).

## 🌐 Hướng dẫn chạy Online bằng GitHub Pages
Ứng dụng có thể được chạy trực tiếp online bằng cách bật tính năng **GitHub Pages** trong phần Settings của Repository này:
1. Vào mục **Settings** của repository trên GitHub.
2. Chọn mục **Pages** ở thanh bên trái.
3. Ở mục **Build and deployment**, phần **Branch**, chọn branch `main` (hoặc `master`) và thư mục `/ (root)`.
4. Nhấn **Save**. 
5. Đợi khoảng 1-2 phút, GitHub sẽ cấp cho bạn một đường link HTTPS công khai. Bạn có thể lưu link này vào điện thoại để mở ra tập luyện mọi lúc mọi nơi!
